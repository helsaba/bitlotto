How the hashes are done for pools

Official pools run by BitLotto will ONLY use the very first tx hash. If the first transactions appear all at once, in one block, it will be the hash in that block that is sorted 0-9, a-f first. The pools total funds are calculated and the hashes are generated just like what would happen if someone made a large payment in the main BitLotto draw address.
