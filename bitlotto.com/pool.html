BitLotto's Stranger Pool

Send multiples of 0.25 to: 1ENjVYVK38aNo1gDiirrcexmT5oq1fmyD (Oct 5)
Each 0.25 multiple will have 10% of the jackpot!!
This pool only accepts multiples of 0.25 BTC. Each group of 10 tickets sorted by block number will share the winnings. How the hashes are done for pools

Official pools run by BitLotto will ONLY use the first valid payment tx hash to the pool address. If the first block in which a pool transaction appears has more than one payment transaction, then the transaction being first will be the first payment transaction hash in that block after sorting all the payment transactions: 0-9, a-f. The pools total funds are calculated and the hashes are generated just like what would happen if someone made a large payment in the main BitLotto draw address.
